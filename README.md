# FrontEnd Pindrop

### Getting Started

- Open a Shell/Terminal/Comand-Line.
- Clone this repository to your machine.
- Go inside of your new folder **`( cd Front-End )`**.
- Switch to my branch **`( git chekout pindrop_requests )`**
- Run **npm install** .
- Run **ng server** .
- Open a browser.
- Go to **( http://localhost:4200/ )** .
